## 1. 概要

### 1.1 目的

本システムは、商談・架電のテキストデータをAI分析し、営業担当者の成長を促す学習プラットフォームを構築することを目的とする。

### 1.2 スコープ

| 対象     | 範囲                            |
| ------ | ----------------------------- |
| 対象ユーザー | REDISH営業担当者、マネージャー            |
| 対象業務   | 架電、商談、学習、分析                   |
| 対象データ  | Google Meet文字起こしテキスト、架電録音テキスト |

### 1.3 前提条件

- Google Meetの文字起こし機能で生成されたテキストデータを使用
- 架電録音の文字起こしは外部サービスで実施済みのテキストを使用
- Gemini APIを使用したAI分析
- 既存のREDISH SFAシステムへの機能追加

### 1.4 制約条件

- Vercelのサーバーレス関数のタイムアウト制限（10秒/無料プラン）
- Gemini APIのレート制限・トークン制限
- Supabaseのストレージ・データベース制限
